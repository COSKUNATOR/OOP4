1) Welche Klasse in Java stellt einen Client-Socket dar, der eine Verbindung zu einem Server über das Netzwerk herstellen kann?

    D) Socket

----------------------------------------------------------------------------------

2) Was ist die Hauptaufgabe der 'ServerSocket'-Klasse in Java?

    C) Sie wartet auf eingehende TCP-Verbindungsanfragen und erzeugt für jede akzeptierte Anfrage einen neuen Socket.

----------------------------------------------------------------------------------

3) Welche Methode verwenden Sie normalerweise auf einem 'ServerSocket', um auf eine neue eingehende Verbindung zu warten und diese zu akzeptieren?

    D) accept()

----------------------------------------------------------------------------------
